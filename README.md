# netinsto

<p><b>A tool made for my NCEA level 2 Computer Science assessment (AS91897, AS91893).</b></p>
<br>
<h1>Q/A</h1>
<p>Q: What is netinsto? <br> A: netinsto is a Deep Packet Inspection (DPI) tool made for inspecting traffic going through your network.</p>
<br>
<p>Q: What can/will netinsto do? <br> A: netinsto has the ability for a user to select a packet and show values such as protocol, source IP, dest IP, source MAC, dest MAC, TTL, src port, dest port, sequence num, ACK num, and finally the application payload. In short form, Layer 2-4 and Layer 7 sniffing. </p>
<br>
<p>Q: What does netinsto mean? It's quite a random name... <br> A: netinsto simply means net(work)ins(pection)to(ol), or Network Inspection Tool. It fits the usecase of the program as it is literally a DPI tool.
